# Graphique

Il existe plusieurs librairies pour afficher des graphiques dans une application Android.
Nous allons utiliser la librairie [MPAndroidChart](https://github.com/PhilJay/MPAndroidChart), mais sachez qu'il existe plusieurs autres librairies qui offrent des fonctionnalités similaires.

Le code de cette recette est disponible [ici](https://github.com/departement-info-cem/3M5-IntroMobile/tree/main/code/ComposeGraphique).

Il s'agit d'une combinaison de recettes précédentes:
- La librairie qu'on utilise est uniquement disponible via JitPack.
- MPAndroidChart fournit des objets graphiques classiques donc il faut utiliser AndroidView pour afficher dans Compose.
